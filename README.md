### How to Use

1. Open `index.html` in your web browser
2. Navigate using the menu bar
3. Switch languages using the language selector (top right)
4. Browse products and pricing plans
5. Contact us through the contact form

### Language Switching

The website automatically remembers your language preference using browser localStorage. Simply select your preferred language from the dropdown, and it will be saved for future visits.
